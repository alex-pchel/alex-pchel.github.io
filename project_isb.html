Interactive Stock Dashboard

Description

A web-based dashboard that provides a company overview, historical share price chart with customizable technical indicators, and a machine learning-based prediction of next-day price movement (up/down). Stock data is fetched via AlphaVantage API, while company descriptions are generated using OpenAI's language model. The price prediction is powered by a classic ML model trained on historical data.
